フロントエンドコース

About

フロントエンドコースでは、webサイトやwebアプリにおいて実際にユーザーの目に触れる部分を作成する方法を学習していきます。
コース内では基本的な事項の紹介にとどめているので、早めに終了して個人開発や案件に挑戦してみてください。

Curriculum

webフロントエンドの基礎から、最新のUIライブラリまでを一通り学習します。
コースを修了すれば案件に挑戦する基礎的なスキルセットが身につくようになっています。

HTML & CSS

HTML（HyperText Markup Language）は、webページの構造や内容を作るためのマークアップ言語です。
CSS（Cascading Style Sheets）は HTML で記述されたページの見た目を整えるためのスタイルシート言語です。
この2つをマスターすることで、基本的なwebサイトが作れるようになります。

Git & Github

gitは、現在最も広く使われているバージョン管理ツールであり、 github は git を用いてソースコードをホスティングするためのプラットフォームです。
現在、チーム開発をする上で git および github の知識は必須のものとなっています。

React

Meta（旧facebook）によって開発されているJavascriptライブラリです。
web上で動作する SPA(Single Page Application) を開発する上でのベースになるほか、ネイティブアプリやモバイルアプリの開発にも応用できるという懐の広さが特徴です。
